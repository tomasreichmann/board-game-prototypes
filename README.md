# board-game-prototypes
Make prototypes for board games using React, TailwindCSS + Daisy UI, super quickly using Vite.

## Prerequisites

```
$ npm install
$ npm install netlify-cli -g
```

## Development

```
$ npm run dev
```

## Deploy

```
$ npm run build
...
$ npm netlify deploy
or
$ npm netlify deploy --prod
```
